Limitaciones conocidas
1

El modelo fue entrenado con un dataset específico y puede no generalizar bien a radiografías con diferentes equipos, técnicas o poblaciones.

La calidad y posición de la radiografía afecta significativamente el resultado.
2

Las imágenes aumentadas sintéticamente para balancear clases pueden no capturar toda la variabilidad clínica real de cada condición.

El oversampling con augmentation es una aproximación, no un sustituto de datos reales.
3

El Grad-CAM muestra qué zonas activaron la predicción, pero no garantiza que sean clínicamente correctas si el modelo tiene sesgos.

Interpretar el mapa de calor con cautela, especialmente en predicciones de baja confianza.
4

La radiografía simple tiene menor sensibilidad que la TC para detectar COVID-19 en fases tempranas.

Una predicción "Normal" no descarta enfermedad en estadios iniciales.
5

Este modelo es un prototipo académico y no ha sido validado clínicamente.

No debe utilizarse para toma de decisiones médicas reales.
